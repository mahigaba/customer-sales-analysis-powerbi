# Customer Sales Analysis Dashboard

## Overview

Interactive Power BI dashboard analyzing customer shopping behavior and sales performance.

This project focuses on cleaning raw data, creating meaningful metrics, and building an interactive dashboard using Power BI.

---

## Tools & Technologies

- Power BI
- Power Query
- DAX
- Python (Google Colab)
- Pandas

---

## Project Workflow

Raw Dataset  
↓  
Data Cleaning (Python)  
↓  
Clean Dataset  
↓  
Data Modeling  
↓  
Power BI Dashboard

---

## Dashboard Features

- Customer distribution analysis
- Sales performance analysis
- Category-wise sales insights
- Customer demographic analysis
- Purchase behavior analysis
- Review rating analysis
- Interactive filters

---

## Key Insights

- Analyzed customer shopping patterns
- Studied sales trends across categories
- Compared customer segments
- Generated business insights from customer data
